# Git branch 전략

## github-flow
- 특별한 이유가 없으면 [github-flow](https://guides.github.com/introduction/flow/) 를 사용한다.

### Branch 역할
- master(main) : 주 Branch
- feature/~ : 기능 추가
- fix/~ : 버그 수정
- refac/~ : 리펙토링
- docs/~ : 문서 추가/수정
